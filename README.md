# ChatGPT-A.I.-Bot-With-Tkinter
Udemy project for learning ChatGPT

## Getting Started

First create a GUI
Second find your API key on https://beta.openai.com/account/api-keys

### Installation
Prepared environment in Visual Studio Code

### Packages

Packages to install

```
pip install tkinter
pip install customtkinter
pip install openai
pip install os
pip install pickle
```
